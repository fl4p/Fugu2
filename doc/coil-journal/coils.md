# Naming Convention
```


KS184µ075s1-CuDHP180s2T24
^ KS=core mat 
  ^ 184=core outer dia (1.84 inch)
      ^ µ075=initial permeability (75µ)
         ^ s1=1 core stacked         
```
```
CuDHP180s2T24
^ wire conductor material name
     ^ conductor diameter (1.8mm) (!= wire diameter !!!)
        ^ s2=2 strands
           ^T24=24 turns
```
